Konfigurationen

zum Ändern der Konfigurationen:
Bitte zuerst den Wildfly Server stoppen: (für Debian) 
service wildfly stop
(für CentOs) 
systemctl stop wildfly
Die Änderungen in der folgenden Datei vornehmen (z.B. mittels nano) 
nano {{properties.aktinPropFile}}
Den Wildfly Service neustarten:(für Debian) 
service wildfly start
(für CentOs) 
systemctl start wildfly
