Hilfe

Tabellarische Daten (z.B. Excel) einfügen

  • Kopieren Sie die gewünschten Daten ohne Kopfzeile aus Excel
    • 1-spaltig ({{reference | patientReferenceToLabel}}), wenn Studien-ID automatisch generiert werden soll
    • 2-spaltig ({{reference | patientReferenceToLabel}} und Studien-ID), wenn Studien-ID manuell eingegeben werden soll
  • Fügen Sie die Daten direkt in das Eingabefeld ein
  • Die Ansicht übernimmt die Struktur der eingefügten Daten automatisch

Zeilen manuell hinzufügen

  • Klicken Sie auf "Zeile hinzufügen", um eine neue Zeile zu erstellen
  • Klicken Sie auf in einer Zeile, um die entsprechende Zeile zu entfernen
  • Tragen Sie die Informationen direkt in die Eingabefelder ein, indem Sie auf die entsprechende Zelle doppelklicken

Hinweise

  • Die Ansicht unterstützt sowohl das Einfügen aus Excel als auch die manuelle Eingabe
  • Klicken Sie auf "Fälle anzeigen", um die Einträge auf Richtigkeit zu überprüfen, falls es nicht schon automatisch geschehen ist
